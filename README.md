## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Maitrak-2661/mathematics_projects.git
   ```

2. Navigate to the project folder:

   ```bash
   cd mathematics_projects
   ```

3. Install required libraries:

   ```bash
   pip install numpy pandas matplotlib seaborn
   ```

4. Run the files:

   * Python scripts:

     ```bash
     python file_name.py
     ```
   * Jupyter Notebook:

     ```bash
     jupyter notebook
     ```
